Test application for Marschpat

Launch app.
You can choose one of two modes - client (main device) or service.

App description for client mode

If you select client mode, you will see next screen:

Here you can connect to device (if you know bluetooth mac-address).
Push at "Connect to device via mac" button. Dialog will appear.

Just input bt mac address and push "OK" button.
Then device will connect to service.
Push button "See device List / connect to device"
You will see list of devices.

Now you can scan for devices.
You need to push "Rescan devices list" button.


An hourglass will be displayed While the client is scans for devices.
You can push "Stop scanning devices" button to stop scanning process.

Device list will be sorted after scan.
You can connect to them and see device info.


Push button "Device details" and you will see this screen.

Here you can see some information about connected device.
For now in device name we use it's MAC address.
Device connection status - (Connected means client device is connected to service)
In message to device field you can enter message to send it to service device. (20 symbols max)
You can push "check" button, or simply push "enter" at keyboard.
Message from service: you can get a message from service device.
Push button "Get message from service" and you will see it.
By default value is "hello".
By pushing refresh button you will refresh next fields: "Device connection state", "Message from device"

App description for service mode

If you select service mode, you will see next screen:

Here you can start/stop service.
Push at "Start service"/"Stop service" button.
You have to start service, if you want to connect to this device.

Push at "See device info" button to see information about device.
Here you can see some information about connected device.
For now in device name we use "Test1". You can simply change it.
Device connection status - (connected/disconnected)
In message to client field you can enter message to write it on service device. (20 symbols max) Client can read this message
You can push "check" button, or simply push "enter" at keyboard.
Message from client: you can get a message from client device.
Push button "Get message from client" and you will see it.
By pushing refresh button you will refresh next fields: "Device connection state", "Message from client"
